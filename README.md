# web3-glue

Development starter kit for web3 node.js apps with nestjs & viem

## Installation

```bash
make
```

## Running the app

```bash
# development
$ make start

# watch mode
$ make start.dev

# production mode
$ make start.prod
```

## Test

```bash
# unit tests
$ make test

# e2e tests
$ make test.e2e

# test coverage
$ make test.cov
```
